## üõ†Ô∏è How to Take Screenshots:

### For Web Screenshots:
- Use your browser's full-page screenshot (Firefox: Shift+F2 ‚Üí `screenshot --fullpage`)
- Or use tools like:
  - **Windows**: Win+Shift+S, Snipping Tool
  - **Mac**: Cmd+Shift+4
  - **Linux**: Flameshot, GNOME Screenshot

### Screenshot Tips:
- Use **clean browser window** (close unnecessary tabs)
- **Remove sensitive data** (real wallet addresses, if any)
- Use **light mode or dark mode** consistently
- Ensure **good contrast** and **readable text**
- Crop to relevant area
- Optimize images (PNG or JPG, < 500KB each)
